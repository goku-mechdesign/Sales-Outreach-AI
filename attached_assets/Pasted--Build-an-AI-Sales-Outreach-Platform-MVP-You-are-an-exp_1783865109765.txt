# Build an AI Sales Outreach Platform (MVP)

You are an experienced full-stack software engineer.

Build a production-quality web application using:

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS
* shadcn/ui
* Neon PostgreSQL
* Prisma ORM
* NextAuth/Auth.js (Google Login)
* Gemini API (Google AI Studio Free Tier)
* Gmail API (Google Workspace)
* Vercel compatible deployment

The application should be modular, cleanly organized, and follow best practices.

---

# Objective

Build an AI-powered outbound sales platform that can:

1. Find prospect companies.
2. Find company contact email addresses.
3. Generate personalized outreach emails in the prospect's language.
4. Send emails from my Google Workspace account.
5. Read replies.
6. Draft replies.
7. Automatically follow up.
8. Notify me when someone is interested.

This is an MVP.

Expected volume:

* 100 outbound emails/month
* 10-20 replies/month

Optimize for simplicity rather than scale.
